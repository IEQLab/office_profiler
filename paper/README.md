# Paper source — SocialSys'26

Submitted in ACM `sigconf` format. The full Overleaf project is reproduced
here verbatim; the ACM template files (`acmart.cls`, the `acm*.bbx`/`acm*.cbx`
biblatex configuration, `ACM-Reference-Format.bst`, `acmguide.pdf`,
`acm-jdslogo.png`, `sample-base.bib`, `acmdatamodel.dbx`) are kept untouched
because the compile graph depends on the exact filenames and contents.

## Build

From this directory:

```sh
pdflatex main-text
biber main-text
pdflatex main-text
pdflatex main-text
```

Or with `latexmk`:

```sh
latexmk -pdf -bibtex- main-text.tex
```

The compile produces `main-text.pdf`.

## Figures

`img/` contains the publication versions of the figures referenced in the
paper. They can be regenerated by running `targets::tar_make()` from the
repository root; the copies under `img/` are checked in so the paper
compiles standalone without running R.
